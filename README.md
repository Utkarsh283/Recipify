# Recipify

Recipify is a web application that displays a random recipe on the home page. Users can search for recipes by name or ingredients, and click on a recipe to view its details.

## Functionality

- Home page displays a random recipe
- Search bar allows users to search for recipes by name or ingredients
- Recipe details page displays recipe image, title, summary, ingredients, and instructions
- Users can navigate between pages using the navigation bar

## Packages Used

- **Vite**: A development server and build tool
- **React**: A frontend JavaScript library for building user interfaces
- **React Router**: A library for managing routes and client-side routing
- **Tailwind CSS**: A utility-first CSS framework
- **PostCSS**: A preprocessor for CSS
- **Autoprefixer**: A plugin for PostCSS that adds vendor prefixes to CSS
- **ESLint**: A linter for JavaScript
- **Axios**: A library for making HTTP requests


## Running the Project

1. Clone the repository using `git clone https://github.com/Utkarsh283/Recipify.git`
2. Install the dependencies using `npm install`
3. Run the development server using `npm run dev`
